El 'main' arranca el programa, y desde este tiene que ser entendido el código.

- Crea tableros (objetos de la clase 'Tablero') para el jugador y la máquina, ubica barcos (objetos de la clase 'Barco')
  en posiciones y orientaciones random.
	
- Las variables constantes que para ello necesita las coge del documento 'variables.py'.
	
- También hay un documento "funciones.py" que tienen funciones (solo una de momento) que no son de ninguna clase pero 
  sirven para simplicar y que quede más limpio el código del main.

Faltan funciones que implementar y el código del "main" para que funcione todo.
